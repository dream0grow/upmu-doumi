## 0. 하는 일

MVP-1(파일명·본문)·MVP-2(마감일) 결과를 모아 **교무수첩 카드 1장**으로 정리합니다.
여기까지 **AI 안 씀** (한줄요약·세부 할일은 MVP-4 에서).

카드 항목:
```
제목 · 발신기관(+등급) · 문서번호 · 본문/첨부
공문성격(5분류) · 처리주체 · 업무유형 · 마감일 · D-day · 놓일 위치
```

## 1. ★ 공문 5성격 자동 분류 (설계서 6장 + 2026-07 사용자 피드백)

```
① 할일형 (보고·제출 필수) → 교무수첩 상단, D-day  (예: 결과 보고, 참석 협조)
② 공람형 (교사 대상 안내) → 공람(담임·교사 안내)  (예: 직무연수 신청 안내)
③ 배포형                → 배포함, 대상 지정      (예: 캠프 포스터)
④ 참고형 (마감일 X)      → 자료실 보관           (예: 통학구역, 첨부 서식)
⑤ 규정형 (마감일 X)      → 읽음표시, 보관        (예: 복무규정 개정)
```

**처리 주체(owner) 힌트**도 함께 붙습니다 (부장이 조정 가능):
- `부장` — 보고·제출·참석 협조 등 부장이 직접 처리
- `담임(공람)` — 담임 선생님들에게 공람으로 알릴 것

**판별 순서** (앞 규칙이 이김 — 규칙 우선, 부장이 조정 가능):
1. **첨부 서식**(`서식1.`/`양식2` 등으로 시작) → **참고형**.
   서식은 본문 공문의 재료라서 **마감일이 있어도 할 일로 올리지 않음**.
2. **참석 협조·협조 요청** → **할일형·부장**.
   ('연수 참석 협조 요청'처럼 연수 키워드가 섞여 있어도 협조 요청이 우선)
3. **보고·제출·조사·수합·현황·실적·회신** → **할일형·부장** (반드시 처리).
4. **이미지 포스터** → **배포형** (학생·가정용 홍보물).
5. **연수·직무연수·신청 안내·참가 신청·모집 안내** 등 교사 대상 → **공람형·담임**.
6. 그 외 **마감일이 있으면 → 할일형·부장**.
7. 마감 없는 나머지를 제목 키워드로:
   - **배포형**: `포스터·홍보·캠프·가정통신·배부·공모전` 등.
   - **규정형**: `규정·법령·예규·조례·훈령·복무·징계·개정령` 등.
   - **참고형**: 위에 안 걸리는 나머지 (통학구역 안내 등).

## 2. D-day 계산

- 오늘 날짜 기준으로 대표 마감까지 남은 일수.
- 표시: `D-7`(7일 남음) / `D-DAY`(오늘) / `D+3(지남)`(3일 지남).

## 3. ★ 오래된 날짜 걸러내기 (실행 마감만)

첨부 서식·계획서에는 **근거 법령일·예시일**(예: 2019·2020년)이 섞여 있습니다.
- **1년(365일) 넘게 지난 날짜는 '실행 기한'이 아니라 참고 날짜로 보고 제외**합니다.
- 덕분에 교무수첩 상단이 옛 근거 날짜로 더럽혀지지 않습니다.
- 단, **올해 안에 며칠~몇 달 지난 마감은 그대로 할일형**으로 둡니다(놓친 것도 알려야 함).

## 4. 업무 유형 (추정 — 부장이 조정)

제목 키워드로 1차 추정: 연수/교육 · 복무/근무 · 행사/사업 · 보고/제출 · 예산 · 규정/법령 · 기타.

## 5. 발신기관 등급 (우선도 힌트 — 설계서 4장)

- **상급기관**: 교육청·교육지원청·교육원·연구원·교육부
- **타학교**: 초/중/고·유치원
- **단체/기타**: 그 외
> 최종 우선순위는 **부장이 직접** 정합니다(설계 원칙). 등급은 참고 힌트일 뿐입니다.
