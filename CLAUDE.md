# CLAUDE.md — 얇은 라우팅

이 프로젝트는 **교무부장 도우미**입니다. 학교 공문을 100% 로컬에서 처리합니다.

## 절대 원칙 (설계서 확정)
1. **로컬 100%** — 데이터가 PC 밖으로 절대 안 나감. 클라우드 API 금지.
2. **정확한 사실은 규칙으로, 창의적 판단은 AI로** — 마감일·발신기관·문서번호는 코드 추출, 요약·할일은 AI 제안.
3. **학생·교사 식별정보 절대 비저장.**
4. **최소 코드** — 과한 추상화 금지. 한 기능씩 완성.

## 개발 순서
- [x] **MVP-1** 파일에서 텍스트 추출 (5형식 + 파일명 파싱) ← **현재 완료**
- [ ] MVP-2 마감일 규칙 추출 + 라벨 구분(신청기한/제출기한/행사일)
- [ ] MVP-3 교무수첩 항목 생성
- [ ] MVP-4 AI 요약·할일 (Ollama, 근거표시 프롬프트)
- [ ] MVP-5 홈 대시보드 (공문 4성격 자동 배치)

## 코드 위치
- `src/parsers/` — 형식별 본문 파서 (pdf·hwp·hwpx·odt·xlsx)
- `src/extractor/` — 규칙 기반 추출 (파일명 파싱). 이후 마감일 추출도 여기.
- `src/extract.py` — 텍스트 추출 총괄 + 명령줄 입구
- `docs/파싱규칙.md` — 5형식 파싱 방법 상세
