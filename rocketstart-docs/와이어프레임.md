# 와이어프레임 — 로켓스타트 화면 5개 (+로그인)

> 이 문서는 로켓스타트 웹 앱의 화면 구조를 글자로 그린 와이어프레임입니다. 모바일 세로 화면(360~420px 폭) 기준으로 그렸고, 데스크톱은 카드가 가로로 넓게 배치된다고 생각하면 됩니다. 실제 화면 이름과 계산식은 `CLAUDE.md`·`IMPLEMENTATION_BRIEF.md` 정의를 그대로 따릅니다.

## 전체 사용자 흐름도

```
[로그인/로그아웃 상태]
      │
      ▼
 로그인 화면 (/login)
      │  (이메일 또는 구글 로그인)
      ▼
 오늘 화면 (/today)  ◄──────────────────────────┐
      │                                          │
      ├─ "새 로켓 만들기" ──► 로켓 만들기 (/new-rocket) ─┤ (저장 후 오늘로 복귀)
      │                                          │
      ├─ 로켓 카드 "집중하기" ──► 집중 타이머 (/timer) ─┤ (세션 종료 후 오늘/리포트로)
      │                                          │
      ├─ 로켓 카드 "복구하기" ──► 복구 모드 (/recovery) ┤ (저장 후 오늘로 복귀)
      │                                          │
      └─ 상단 내비 "리포트" ──► 리포트 (/report) ──────┘

공통 내비게이션(상단, 모든 화면 공통):
[오늘] [로켓 만들기] [타이머] [복구] [리포트]  ...  [로그아웃]
```

## 2. 로켓 만들기 (/new-rocket)

> 프로젝트 · 로켓 목표 · 90% 산출물 체크리스트 · 산출물별 하위과업 · 하위과업별 예상 뽀모도로 · 로켓 시작일 · 3일 계산 방식을 입력합니다. 90% 목표일과 총 예상 뽀모도로는 자동 계산됩니다.

```
┌─────────────────────────────────┐

│                                  │
│ 프로젝트명                          │
│ ┌────────────────────────────┐  │
│ │ 예: 로켓스타트 웹 MVP          │  │
│ └────────────────────────────┘  │
│                                  │
│ 로켓 태스크명                       │
│ ┌────────────────────────────┐  │
│ │ 예: 오늘 화면과 타이머 MVP 만들기│  │
│ └────────────────────────────┘  │
│                                  │
│ 3일 안에 보여줄 90% 산출물           │
│ ┌────────────────────────────┐  │
│ │ (여러 줄 입력)                │  │
│ └────────────────────────────┘  │
│                                  │
│ 로켓 시작일        3일 계산 방식     │
│ ┌───────────┐   ┌─────────────┐ │
│ │ 2026-07-01 │   │ 주말 포함 ▾  │ │
│ └───────────┘   └─────────────┘ │
│                                  │
│ ── 90% 산출물 체크리스트 ──────────  │
│ ┌───────────────────────────┐   │
│ │ 90% 산출물 기준 1             │   │
│ │ 필수 ☑   가중치 [50]          │   │
│ │  ┗ 하위과업                  │   │

│ ┌───────────────────────────┐   │
│ │ 90% 산출물 기준 2             │   │
│ │ 필수 ☑   가중치 [50]          │   │
│ └───────────────────────────┘   │
│ [+ 산출물 추가]                   │
│                                  │
│ ── 자동 계산 ─────────────────────  │
│ 90% 목표일        2026-07-03     │
│ 총 예상 뽀모도로    6              │
│                                  │
│ ┌────────────────────────────┐  │
│ │   로켓 발사하기 (button)      │  │
│ └────────────────────────────┘  │
└─────────────────────────────────┘

## 3. 집중 타이머 (/timer)

> 15분 최소 시작 / 25분 기본 / 50분 깊은 작업 / 사용자 지정. 백색소음(화이트/브라운/빗소리/카페/무음). 완료 세션은 실제 뽀모도로로 기록됩니다. 어느 로켓을 집중할지 선택합니다.

```
┌─────────────────────────────────┐
│ 집중 타이머                        │
│ 완료한 세션은 실제 뽀모도로로 기록돼요 │
│                                  │
│ 어떤 로켓에 집중할까요?              │
│ ┌────────────────────────────┐  │
│ │ 로켓스타트 웹 MVP 만들기 ▾     │  │
│ └────────────────────────────┘  │
│                                  │
│ 시간 선택                          │
│ ┌───────┐┌───────┐┌───────┐     │
│ │15분 시작││25분 기본││50분 깊게│     │
│ └───────┘└───────┘└───────┘     │
│ ┌────────────────────────────┐  │
│ │ 직접 입력 [   ]분             │  │
│ └────────────────────────────┘  │
│                                  │
│         ┌─────────┐             │
│         │  24:31   │             │
│         │ (원형 진행) │             │
│         └─────────┘             │
│                                  │
│   [일시정지]      [종료하고 기록]    │
│                                  │
│ 백색소음                           │
│ ○화이트 ○브라운 ○빗소리 ○카페 ●무음  │
└─────────────────────────────────┘
```

**세션 완료 직후 상태**
```
┌─────────────────────────────────┐
│        ✅ 세션 완료!               │
│   25분 집중, 1뽀모도로로 기록했어요   │
│                                  │
│  이번 세션 메모 (선택)               │
│ ┌────────────────────────────┐  │
│ │                              │  │
│ └────────────────────────────┘  │
│  ┌───────────┐ ┌─────────────┐  │
│  │ 오늘로 가기  │ │ 한 세션 더   │  │
│  └───────────┘ └─────────────┘  │
└─────────────────────────────────┘
```

**오류 상태 (로켓 선택 안 함)**
```
│ ⚠ 먼저 집중할 로켓을 선택해 주세요    │
```

## 5. 리포트 (/report)

```
┌─────────────────────────────────┐
│ 계획 정확도 리포트                  │
│ 산출물 달성률과 뽀모도로 예측 정확도를 │
│ 분리해서 봅니다.                    │
│                                  │
│ ┌───────────────────────────┐   │
│ │ 로켓스타트 웹 MVP 만들기      │   │
│ │ 산출물 달성률        85%      │   │
│ │ 예상 뽀모도로         4       │   │
│ │ 실제 뽀모도로         5       │   │
│ │ 사용률              125%     │   │
│ │ 예측 정확도          75점     │   │
│ │ 판단          [과소예측]      │   │
│ └───────────────────────────┘   │
│                                  │
│ ┌───────────────────────────┐   │
│ │ 동아리 계획서                 │   │
│ │ 산출물 달성률        0%       │   │
│ │ 예상 뽀모도로         3       │   │

└─────────────────────────────────┘
