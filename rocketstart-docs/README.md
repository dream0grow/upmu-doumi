## 핵심 개념 3줄

1. **90% 완료**는 뽀모도로 사용량이 아니라 **필수 산출물 체크리스트 달성률**로 판단한다.
2. **뽀모도로**는 완료 기준이 아니라 **계획 예측 정확도를 학습**하는 데이터다.
3. **미루면** 미완료로 방치하지 않고 **복구 질문**으로 전환한다.

데이터 흐름: `Project → RocketTask → Deliverable → Subtask → FocusSession` (+ `ReflectionLog`)
